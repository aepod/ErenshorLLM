# A Bounty on Ripper DeMerth

Back to Quests

To begin this quest, prompt Captain Kilkay in Port Azure with Ripper's. It is completed by handing him the Ear of Ripper.

Completing stage 1 of this quest drops your reputation with the Ripper DeMerth faction to hostile levels.

It is currently impossible to complete stage 2 of this quest.

Stage: Name: Description
1: A Bounty on Ripper DeMerth: Guard Captain Kilkay in Port Azure has offered a reward for the death of Ripper DeMerth, a traitor to Port Azure.

Return with proof of the demise of Ripper.
