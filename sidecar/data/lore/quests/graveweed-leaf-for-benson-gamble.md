# Graveweed Leaf for Benson Gamble

Back to Quests

This quest given to Paladins only. To start the quest speak to Benson Gamble in Hidden Hills and prompt him with prove.

Any class can complete handing Graveweed Leaf to Benson Gamble.

Graveweed Leaf is dropped from Neophyte Necromancer or Zashlyn Bloodbane.

Stage: Name: Description
1: Graveweed Leaf for Benson Gamble: Benson Gamble in Hidden Hills is looking for Graveweed Leaf from the nearby Bonepits.
