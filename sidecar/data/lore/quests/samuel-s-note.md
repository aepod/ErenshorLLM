# Samuel's Note

To start this quest say note to Samuel Thatcher in Hidden Hills. You can complete it by handing Samuel's Note to Harvey Alder in Port Azure

Stage: Name: Description
1: Samuel's Note: Samuel in Hidden Hills has asked you to deliver a note to his good friend Harvey Alder, who is likely staying at the Azure Arms in Port Azure

Completion Message:

Ah, Samuel is a good friend. Thank you for bringing this to me. I don't have much, but take this as a token of my gratitude... and welcome to Port Azure.
