# A Bounty on Guard Captain Kilkay

Back to Quests

To start this quest speak to Ripper DeMerth in Ripper's Keep with the prompt Kilkay

Completing stage 1 of this quest drops your reputation with the Azure Guard to hostile levels.

It is currently impossible to complete stage 2 of this quest.

Stage: Name: Description
1: A Bounty on Guard Captain Kilkay: Ripper DeMerth wants you to kill Guard Captain Kilkay in Port Azure, for his role in the abandonment of the Citizens of the North.

Return with proof of his demise.
