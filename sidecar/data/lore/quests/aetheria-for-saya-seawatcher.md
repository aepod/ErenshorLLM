# Aetheria for Saya Seawatcher

Back to Quests

To start the quest speak to Saya Seawatcher in Vitheo's Watch and prompt her with Aetheria.

This is a repeatable quest.

Aetheria is dropped from ghosts, to list some: A Restless Spirit, A Sorrowful Spirit, Mourning, A Lost Sea Giant and The Lost Sea Giant King.

Stage: Name: Description
1: Aetheria for Saya Seawatcher: Ghostly beings of Erenshor can leave behind Aetheria if you destroy them. If you find any, Saya Seawatcher in Vitheo's Watch says she has a unique reward for you.
