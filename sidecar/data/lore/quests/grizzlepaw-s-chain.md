# Grizzlepaw's Chain

# Grizzlepaw's Chain

Back to Quests

This quest is given to stormcallers, but can currently be started by any class. To start the quest, speak to Windlae Oldseer in Hidden Hills and prompt her with test.

Any class can complete this quest by handing Windlae Oldseer a Weathered Chain

Weathered Chain is dropped from Grizzlepaw inside Faerie's Brake.

Stage: Name: Description
1: Grizzlepaw's Chain: A Stormcaller named Windlae in Hidden Hills has asked that you find and fell a beast in the Brake called Grizzlepaw.
She has observed a remnant of a chain stuck in the beast's teeth that may have belonged to a friend of hers.

On Completion

Ah, so it is true then. Tragedy has befallen my friend but at least revenge has been taken. Thank you for your services, please take this as a token of my appreciation.
