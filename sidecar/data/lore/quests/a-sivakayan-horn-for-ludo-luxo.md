# A Sivakayan Horn for Ludo Luxo

Back to Quests

This quest given to Duelist only. To start the quest speak to Ludo Luxo in Hidden Hills and prompt him with task.

Any class can complete this quest by handing Ludo Luxo a Sivakayan Horn

Sivakayan Horn is dropped from A Lesser Avatar of Sivakaya inside The Bone Pits.

Stage: Name: Description
1: A Sivakayan Horn for Ludo Luxo: Ludo Luxo in Hidden Hills has asked for you to find a Sivakayan Horn from the Bonepits. They make great sharpening stones.
