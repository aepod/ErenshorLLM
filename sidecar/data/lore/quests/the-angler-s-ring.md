# The Angler's Ring

Back to Quests

To start and complete the quest, hand the Angler's Ring to Liani Bosh

Prompt Liani Bosh with help to obtain Mold: Angler's Ring

Note: while the quest completion is not repeatable, you can obtain additional Molds for the Angler's Ring if you have deleted or used the previous one.

Once this quest is completed, you can travel to Blacksalt Strand and find Bassle Wavebreaker at night to complete Meet The Fisherman and return to Stowaway's Step.
