# Reliquary Hall

# Reliquary Hall

## Reliquary Hall

### Type

Zone

### Level

16+

The Reliquary Hall is a customizable "Guild Hall" accessed through Port Azure.

To access the Reliquary Hall, talk to the Reliquary Guard to receive the  Reliquary Key that opens the nearby Reliquary Gate. Walk through the door beyond that gate to enter the Reliquary Hall.

Upon your first visit, you'll need to defeat Reliquary Fiend to gain full access to the features of the Reliquary Hall. This enemy will not respawn, and does not need to be defeated on any alt characters; once one of your characters has defeated it, you will permanently have full access.

In the far end of the hall is a podium. Click it to gain access to the customization features and grab the  Box of Portals, an item that will teleport you back to the Reliquary Hall.

Bosses
Reliquary Fiend

## Reliquary Customization

The Reliquary Hall's customization menu has 122 customization slots, plus one slot that stores the  Box of Portals. The  Box of Portals slot should be left empty so that new copies of the item can be picked up by alt characters. Once you have slotted in items, you need to press the "build" button to get them to show up in the Reliquary Hall.

The four outermost slots (Willow, Celestial, Honor, Elements) are for the corresponding raid Runes. Raids are not yet implemented, but you can already acquire these runes and place them in the Reliquary in order to be prepared with portals to enter them upon their eventual release.

The next slots are four dedicated slots for statues. A list of the statues and where to acquire them is listed in a table below.

The other six slots in the same section are for non-raid Runes. Slotting in the 6 runes into these slots will create portals along the outside walls of the Reliquary to access the corresponding zones. This allows you to have easy access to these runes by only carrying around the  Box of Portals to return to the Reliquary Hall. There is one additional portal in the Reliquary hall up the stairs at the far end of the hall; this will return you to wherever you were last before teleporting to the Reliquary Hall by using the  Box of Portals.

The eight innermost slots are for "decor" items. Most decor items are cosmetic, although some offer useful quality-of-life features.

The majority of the decor items are acquired through interactions with Breena Carpenter, the furniture vendor in Port Azure. Breena starts off with a limited set of furniture items for purchase, but more can be acquired through giving her blueprints. Blueprints are acquired as random drops from enemies in Loomingwood, Rockshade Hold, Azynthi's Garden. Once she is given the appropriate blueprint, the item will be available for purchase from her.

Additionally, some furniture items drop directly in a usable form without needing to use Breena Carpenter. These drop in Fallen Braxonia, Blooming Sepulcher, Soluna's Landing, and Vitheo's Rest. ## Furniture Quality-of-Life Features

A few of the furniture items available for your Reliquary Hall offer useful quality-of-life features to your Reliquary Hall and are not solely cosmetic.

**Banks** :  Wood Bank Set,  Loomingwood Market Set,  Stone Bank Set

**Auction Houses** :  Wood Auction Set,  Loomingwood Market Set,  Stone Auction Set

**Training Dummies** :  Wood Training Set,  Stone Training Set

**Vendors** :  Wood Vendor Set,  Stone Vendor Set

**Proficiency Respec** :  Wisp Garden

**Braxonian Flame Well** :  Flame Well Seed (allows upgrading Item Quality to blue without

Note that  Loomingwood Market Set is correctly listed as both a bank and an auction house; using it does allow you to get both while using up only one decor slot.
