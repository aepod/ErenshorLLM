# Location

A Deepling Crimsonite can be found throughout the central caves of the Elderstone Mines.

# Appearance

A Deepling Crimsonite is a serpentine enemy wearing bangles on each wrist and a belt. As the name implies, it is deep red in color. It has sharp claws and a forked tongue.

# Behavior

A Deepling Crimsonite roams the caves and will attack players if they get too close.
