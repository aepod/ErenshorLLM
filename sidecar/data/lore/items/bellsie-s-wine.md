# Bellsie's Wine

## Bellsie's Wine

### Type

Quest Item

### Reward from

The Mystery of Bellwain Island

**Bellsie 's Wine** is a Quest Item in Erenshor.

# Location

Bellsie's Wine is found in a cavern beneath the docks during The Mystery of Bellwain Island. Its location is hinted at by the note  The Stash.

# Description

Bellsie's Wine can be delivered to the empty seat near the campfire behind the pergola, as instructed by the note  Relaxation. Upon doing so,  Afternoon on the Isle will appear.

Bellsie's Wine

A fine wine from Merosavilla, very aged and probably delicious.
