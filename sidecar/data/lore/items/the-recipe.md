# The Recipe

# The Recipe

## The Recipe

### Type

Quest Item

### Related quest(s)

The Mystery of Bellwain Island

**The Recipe** is a Note in Erenshor.

# Location

This note can be found on Bellwain Island as part of the quest The Mystery of Bellwain Island.

# Description

The Recipe details how to make  Breaded Island Fin.

The Recipe

Written in decidedly feminine handwriting:

Picnics do not get any better than some brackfruit-battered Island Fin!

-Island Fin Fish
-Brackfruit juice
-Bread
-1 Coal

Once the four ingredients are given to the Baking Oven in the kitchen, the resulting Breaded Island Fin can be brought to the picnic blanket where it will be consumed and the note  Mid Day on the Isle will appear.
