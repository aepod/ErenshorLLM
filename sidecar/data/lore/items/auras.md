# Auras

# Auras

Auras are buffs that apply to both the players and their squad mates. There are both class-specific auras and other auras that can be used by multiple classes. The first class specific aura is given to that class on Stowaway's Step on character creation as part of their starter equipment, level 2 class auras are sold by class vendors in Port Azure, level 3 and 4 class auras are found from killing enemies.
