## Mooring Coin

### Type

### Reward from

**Mooring Coin** is a Teleportation Item in Erenshor.

# Location

Mooring Coin is the reward given for completing The Mystery of Bellwain Island. It can be found in Bellwain's Master Bedroom after obtaining  Bellwain's Master Key.

# Description

Mooring Coin teleports the user back to the last Wishing Well the player saved at. It can either be right clicked to use it from the inventory, or it can be placed into a hotkey slot and used by pressing the corresponding key.
