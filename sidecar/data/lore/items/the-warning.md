**The Warning** is a Note in Erenshor.

The Warning

Should any set foot in this home, I implore you to return from where you came. It holds loss and sorrow and hopelessness - all of the wealth i have sailed for is worthless alone.

# Location

This note can be found on Bellwain Island as part of the quest The Mystery of Bellwain Island.

# Description

The Warning is a note left by Captain Bellwain for any who might visit his estate after his departure.

The Warning
Should any set foot in this home, I implore you to return from where you came. It holds loss and sorrow and hopelessness - all the wealth I have sailed for is worthless alone.

My love, my light, my Bellsie is no more, and so shall I return to the sea one last time.

I have narrated our lives here if you are inclined to stay, please honor our memories.

-Bellwain

This note is purely story-related and does not offer any puzzle hints.
