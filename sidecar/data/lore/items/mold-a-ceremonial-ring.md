# Mold: A Ceremonial Ring

## Mold: A Ceremonial Ring

### Type

Mold

### Other sources

Fishing

This Mold can be found in Fernalla's Revival Plains by fishing, it can be fished at any time of the day. To craft the normal  Ceremonial Ring you need to use  Coal as a fuel source, but if you want to craft the  Pristine Ceremonial Ring you need to use  Ancient Coal as a fuel source.

Mold: A Ceremonial Ring

Ingredients:

1x  Chunk of Copper Ore
1x  Sivakayan Horn
1x  Luminstone
1x  Sivakrux

Creates:

Ceremonial Ring

Note: Ingredients MUST be exact quantities

Use CTRL + CLICK to separate stacks.
