## Bellwain Bedroom Key

### Type

Key, Quest Item

### Related quest(s)

**Bellwain Bedroom Key** is a Quest Item in Erenshor.

# Location

Bellwain Bedroom Key is found on the roof of the house as part of the quest The Mystery of Bellwain Island. Its location is hinted at by the note  Detective Books.

# Description

Bellwain Bedroom Key is used to open the first door on the left of the long hallway on Bellwain Island.
