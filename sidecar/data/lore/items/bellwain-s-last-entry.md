# Bellwain's Last Entry

# Bellwain's Last Entry

## Bellwain's Last Entry

### Type

Quest Item

### Related quest(s)

The Mystery of Bellwain Island

**Bellwain 's Last Entry** is a Note in Erenshor.

# Location

Bellwain's Last Entry is the final note found on Bellwain Island as part of the quest The Mystery of Bellwain Island.

# Description

Bellwain's Last Entry describes Bellwain's departure from the island.

Bellwain's Last Entry

This is not a place to be alone. It is with age and sorrow that I leave tomorrow for one final voyage to the sea. I will not return to this place, for nothing holds me here. I've given back the materials, and now I give myself back to this world.

I can only hope to be reunited with Bellsie one day, but alas, I do not deserve such a kind fate.

Returning home can't always be the same gift that it was the day before.

It does not offer any clues, but rather serves as the final chapter of the story.
