# Bellwain's Master Key

## Bellwain's Master Key

### Type

Key, Quest Item

### Reward from

The Mystery of Bellwain Island

**Bellwain 's Master Key** is a Quest Item in Erenshor.

# Location

Bellwain's Master Key is received upon turning in the letters as part of the quest The Mystery of Bellwain Island.

# Description

Bellwain's Master Key is used to open the door to the master bedroom of the house on Bellwain Island.
