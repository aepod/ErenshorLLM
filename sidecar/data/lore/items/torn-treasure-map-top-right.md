## Torn Treasure Map (Top Right)

### Type

Quest Item

### Other sources

All Enemies have a 1.25% chance to drop one of the four map pieces as a global drop.
All Fishing locations have a 5% chance to catch one of the four map pieces.

### Related quest(s)

Repairing Treasure Maps

Can appear when fishing in Faerie's Brake.

If all 4 pieces are handed to Cecil Threbb in Faerie's Brake they can be combined to create a Treasure Map.
