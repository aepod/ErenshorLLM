# Chat Commands and Controls

Erenshor uses chat commands for communication and SimPlayer interaction. These commands are essential for grouping, trading, and navigating the game world.

---

Communication commands: "/whisper [name] [message]" sends a private message. "/group [message]" sends to party chat. "/shout [message]" broadcasts zone-wide. "/players" lists players in the current zone. "/all players" shows the global player list. "/time" shows current game time. "/loc" displays player coordinates.

---

SimPlayer recruitment: "/whisper [name] let's group!" invites a specific SimPlayer (must be within 3 levels). "/shout LFG!" broadcasts a looking-for-group request with approximately 20% response rate. Naming specific players guarantees responses. Greetings trigger 40-50% response rates; mentioning a player's name guarantees 100%.

---

Group commands: "/group attack" orders party to engage. "/group pull" has the puller initiate combat. "/group guard" sets defensive posture. "/group flee" orders retreat. Only one player can hold Tank, Assist, or Puller roles simultaneously. The auto-pull system considers max/min enemy levels, pull distance (approximately 30 meters for spells), and group mana percentage.

---

Friend system: "/friend [name]" binds a SimPlayer for level-matching. This ensures the SimPlayer stays at an appropriate level for grouping. The "/guildinvite" command manages guild invitations.

---

Controls: WASD for movement, Spacebar to jump, right-click for camera, I for inventory, B for spell/skill book, J for quest journal, left-click to target, Q to attack, left-click to interact with doors/NPCs/shops. Ctrl+Left-Click splits stacks to singles, Shift+Left-Click creates groups of 10.

---

Save system: Always disconnect properly using the in-game option. Auto-saves occur when opening/closing inventory, changing equipment, or zoning. The game maintains 10 backup saves. Right-click shopkeepers to buy/sell. Right-click items to consume ability books.
