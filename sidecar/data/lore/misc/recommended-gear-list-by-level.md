This page is a list of potentially viable gear at various levels. Please see Recommended Gear for starter and pre-raid recommendations.

This list provides a list of item names, the wearable location, and the recommended player level.

Note that the recommended player level is a level suggesting the player should be able to obtain the gear at or by this level. Not the level of the item specifically.
