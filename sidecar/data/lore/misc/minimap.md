# Minimap

## Minimap

The minimap provides an overhead view of the area surrounding the player as well as providing information about NPCs around the player. When the minimap is enabled on the Server Admin Panel, it will appear in the upper right corner of the screen below the option buttons by default. It can be expanded to a larger view by pressing the Map button ( **M** by default). Upon entering a zone, the minimap will be covered in a black fog of war. The player must explore the zone to remove these black areas before they will be visible. Once uncovered, the areas will be available to all player characters on the server (save file).

### Options

There are five buttons beneath the minimap that control its functionality. These options are not currently saved at logout and will need to be changed again when players log into the game.

Minimap Options  Icon: Name: Description
+: Zoom In: Zooms the camera closer in the minimap
-: Zoom Out: Zooms the camera further out in the minimap
V: Draw real-time geometry: Causes the minimap to render the actual game world rather than using pre-built images. Can negatively impact FPS
M: World Map: Opens the world map
^: Lock North: Forces the minimap to always orient north. By default it rotates with the player.

### NPC Dots

The minimap updates in real time to display indicators of nearby NPCs. These dots are color-coded to give players an indication of the NPC's hostility.

Dot color: Meaning
Green: The Player
Blue: Simulated Players
Red: Hostile Enemy
White: Neutral NPC/Enemy
Yellow: Quest NPC

### Treasure Markers

If a player has an active treasure chest in the current zone, it will appear as an orange **X** on the minimap as well.
