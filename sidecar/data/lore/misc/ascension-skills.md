# Ascension Skills

# Ascension Skills

## Ascensions

Ascension skills provide an additional level of progress after the character reaches level 35. Each ascension level requires 38,000 XP to complete, resulting in one ascension point, and there is no cap meaning you can eventually fill out the entire tree. Simulated Players receive ascension points as well and allocate them automatically without input from the player. You may view a simulated player's ascensions by targeting them and typing `/ascview`, but they cannot be changed. Ascension points cannot be respecced, but this should not be an issue since there is no cap.

## General Skills

These skills are available to all classes and provide stat boosts. These boosts are applied after all other buffs, equipment values, etc.
