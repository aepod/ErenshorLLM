# Server Admin Panel

The Server Admin Panel

## Server Admin Panel

The server admin panel allows players to customize their server settings to tailor the game to their own play style. These options affect all player characters in the server (the save file) and can be changed at any time with no penalty. The server admin panel is accessed by pressing the "View Server Admin Panel" button at the bottom of the login screen.

The configuration affects how the player themselves interact with the game, rather than affecting simulated players directly unless the option specifies "global" such as the run speed modifier. This means that simulated players will not receive bonus XP themselves, but they will still stay in similar ranges to the player via the normal catch-up mechanics.

## Options

There are numerous options that can be configured individually as well as preset buttons for different configurations.

### XP Penalty On Death

When enabled, causes the player to lose 33% of the experience points needed to level on death. This XP loss cannot lower a player's level or ascension level.

### Other Options

Below the checkboxes, there are several sliders for configuring additional gameplay options.

Respawn Time Modifier: Affects the respawn rates of all NPCs. Lower numbers cause faster respawns: 0.25: 20.00: 1.00: 0.80: 2.00: 1.00

#### Notes

Drastically increasing the XP modifier will cause players to level extremely quickly. This can cause situations where players have out-leveled a particular area, but have not received enough loot to gear up their party.

Higher values of the rare drop modifier may cause a player to receive an excess of gear, quickly filling their inventory and requiring more frequent trips to a vendor or the auction house.

You can see your configuration in game by typing `/ruleset` into chat.
