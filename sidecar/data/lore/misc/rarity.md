# Rarity

# Rarity

There are 4 different rarities in the game.

* Common: 70%
* Uncommon: 15%
* Rare: 5%
* Legendary: 1.3%
* nothing: 8.7%

The way the drop system works is when an enemy dies a d100 is rolled, if the value is less than legendary rate, legendary drops, if it's not less than legendary it checks against rare, then uncommon, etc. if it's not even less than common, it nothing drops.
